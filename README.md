## Project Objective
To analyze engagement across social media platforms by studying likes, shares, comments, post types, monthly activity, and user sentiment, helping understand audience behavior and content performance.

## Questions
- Which platform generates the highest positive, negative, and neutral sentiments?
- What post type gets the most likes, comments, and shares?
- Which days and months have the highest posting activity?
- Which platform shows consistent engagement throughout the year?
- What is the overall engagement status (total likes, shares, comments)?

## Process
- Imported dataset into Power BI
- Cleaned and formatted dates, platform names, and sentiment labels
- Created DAX measures: total likes, shares, comments
- Built visuals: bar charts, donut charts, area chart, comparison chart
- Added slicers for sentiment, platform, day, and post type
- Designed dashboard focusing on user-friendly layout and insights

## Project Insights
- Total engagement includes 2,38,181 likes, 41,565 shares, and 20,266 comments.
- Instagram and Twitter show higher positive sentiment compared to Facebook.
- Video posts receive the highest likes and engagement.
- Posting activity peaks in May and dips in July.
- Saturday and Friday have the highest number of posts by day.

## Final Conclusion
The dashboard effectively highlights how sentiment, post type, and timing influence engagement across platforms.
